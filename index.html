A discussion about fonts:

Urdu literature is best enjoyed using a dedicated. We recommend Jameel Noori Nastaleeq which you can download here. You will have to install this font in your e-reader and select it when reading books written in Urdu. In the Kindle this can be achieved by creating a fonts folder in the root of the device (if it doesn't already exist) and copying the font file inside it.
