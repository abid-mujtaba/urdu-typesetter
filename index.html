A discussion about fonts:

The embedded font is Jameel Noori Nastaleeq (TTF) which is 11 MB in size. The ebook variants without an embedded font are considerably smaller in size but require that the font be installed in the device separately. In the Kindle this can be achieved by creating a fonts folder in the root of the device and copying the font file inside it.
